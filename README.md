# 🌌 Singulo: The Knowledge Big Bang
> **Stop Scrolling. Start Exploring. Your Private Universe Awaits.**

**Singulo** 是一款颠覆传统的“主动型”双生 AI 资讯引擎。它彻底抛弃了瀑布流式的图文堆砌，将信息的获取转化为一场跨越 137 亿年时空的“知识大爆炸”。

---

## 💡 核心哲学：从“投喂”到“演化”

在 **Singulo** 中，信息不再是被动等待点击的死板链接，而是不断生长的**兴趣星系**。

1. **从奇点开始 (The Singularity)**：
   一切始于黑暗中的一个发光点。输入你的意图，长按充能，松开瞬间，AI 将为你炸裂出一个独一无二的 3D 知识宇宙。
2. **双生 AI (The Twin Intelligence)**：
   它不是一个冷冰冰的聊天窗口，而是你的数字孪生。它比你更博学，比你更敏锐，它在后台持续猎取全网资讯，主动为你扩展认知的边界。
3. **空间化探索 (Spatial Exploration)**：
   利用 Web3D 与 GPU 渲染技术，信息以星系、星云和节点的形式分布。当你靠近，知识如同树根般分叉生长；当你远离，它在星空深处静候你的下一次造访。

## 🚀 关键特性 (Core Features)

### 💥 仪式感：大爆炸交互
* **充能触发**：模拟宇宙大爆炸的物理过程，通过长按空格/屏幕积蓄能量，视觉与音效的极致配合打造“第一眼冲击力”。
* **动态演化**：基于 D3-Force 算法与 Three.js，节点在爆炸后自发形成逻辑关联的星团。

### 🤖 智脑：主动式双生 Agent
* **深度记忆**：基于向量化用户画像，Singulo 记住你的每一次偏好，过滤噪音，只保留能促进你潜力发展的“高质量物质”。
* **自主分销**：采用类似多专家（MoE）架构，不同领域的子 Agent 协作生成摘要、对比与分析。

### 🌿 生长：动态 LOD 节点
* **无限细分**：靠近节点时，AI 实时生成深层关联内容。
* **媒介自适应**：系统自动判断内容属性，动态匹配最适合的呈现媒介（从极简图文到 3D 交互模型）。

### 🌀 跃迁：虫洞机制
* **打破茧房**：当你陷入单一兴趣点时，触发“虫洞”进行狂野的跨界联想，带你降落在完全陌生但极具启发性的新星系。

## 🛠️ 技术架构 (Tech Stack)

| 维度 | 技术选型 |
| :--- | :--- |
| **前端渲染** | React + React Three Fiber (R3F) + Three.js |
| **物理引擎** | D3-force-3d |
| **后台引擎** | Node.js (Codespaces 原生开发) |
| **AI 模型** | LLM Orchestrator (调度者) + Multi-Worker Agents |
| **存储/记忆** | Local Vector Store + Persistent JSON DB |
