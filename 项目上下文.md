# Singulo 项目开发上下文 (Project Context)

## 1. 产品愿景 (Product Vision)
**Singulo** 是一款空间化、主动化的“双生AI”资讯与知识探索引擎。
不同于传统的 2D 瀑布流图文平台（如 particle.news），Singulo 将信息获取转化为 3D 宇宙中的“大爆炸”与“星系探索”。它通过一个懂用户的后台 Agent 持续检索全网信息，并以 3D 节点树的形式动态生成和呈现知识脉络。

## 2. 核心体验机制 (Core Mechanics)
* **奇点爆炸 (The Big Bang):** 初始全黑屏幕，用户输入一个兴趣词（如“赛博朋克”），长按充能，松开后全屏粒子爆炸，瞬间展开为一个由 AI 生成的 3D 知识拓扑图（兴趣星系）。
* **双生 Agent (Twin AI):** 后台记忆用户的探索路径。基于 RAG/本地向量池，实现 80% 迎合兴趣 + 20% 强制跨界探索（打破信息茧房）。
* **动态视距与生长 (LOD & Branching):** * 采用**固定视角**，鼠标拖拽旋转整个 3D 宇宙（历史节点在左上角，新节点在四周）。
    * 点击/靠近某个节点时，触发后台 AI 实时生成（Lazy Load），节点像树根一样分叉长出子节点。远离的节点模糊或冻结以节省性能。
* **媒介自适应 (Media Routing) [MVP阶段简化]:** 终极形态是 AI 决定输出 3D模型、视频或图文。**在目前的 14 天 MVP 阶段，统一降级为 Markdown 图文呈现**，保留 API 路由框架，后期接入多模态。
* **虫洞跃迁 (Wormhole):** 探索疲劳时，一键触发跨界联想，生成完全不同但底层逻辑相关的全新星系。

## 3. MVP 阶段严格限制 (14-Day Sprint Constraints)
为了保证两周内上线，请 Code Agent 在生成代码时**严格遵守以下边界**，切勿过度工程化：
1.  **不做全自由视角的 3D 漫游**，使用 `OrbitControls` 限制在固定主视角，通过旋转查看全局。
2.  **不做复杂的媒体流生成**，所有 AI 节点生成的子内容目前只返回结构化的 JSON 和 Markdown 文本。
3.  **不做复杂的云端数据库**，初期用户画像和历史探索路径（双生记忆）直接存在浏览器 `LocalStorage` 中。

## 4. 技术栈选型 (Tech Stack)
| 模块 | 推荐技术/库 | 说明 |
| :--- | :--- | :--- |
| **前端框架** | React | 组件化构建 UI 抽屉和交互逻辑。 |
| **3D 渲染** | React Three Fiber (R3F) + Three.js | 实现粒子大爆炸、发光节点、连线。 |
| **物理布局** | d3-force-3d | 用于计算节点爆炸后的排斥力，防止 3D 空间重叠。 |
| **后端服务** | Node.js (Express) | 与前端语言统一，适合 Github Codespaces 云原生开发。 |
| **AI 调度** | LLM API (支持 Structured Outputs) | 后端采用 Orchestrator (路由) + Workers (生成) 架构。 |

## 5. 核心数据结构约束 (Core Data Schema)
Code Agent 在设计前后端交互时，核心应当围绕这棵“动态 JSON 树”展开。每个 Node 的基础结构建议如下：
```json
{
  "id": "uuid",
  "parent_id": "uuid_or_null",
  "topic": "节点核心词",
  "summary": "简短的hover摘要",
  "content": "具体的Markdown长文本或过渡内容",
  "status": "frozen | active | exploring",
  "media_type": "text" // 为后续多模态预留扩展字段
}
```

## 6. 开发节奏指南 (For Code Agent)
* Step 1: 不做全自由视角的 3D 漫游**，使用 `OrbitControls` 限制在固定主视角，通过旋转查看全局。
* Step 2: 构建前端 R3F 场景，实现输入框 -> 粒子爆炸 -> D3力导向图渲染 3D 节点。
* Step 3: 实现点击交互：点击节点 -> 触发后端生成子节点 -> 3D 连线动态生长 -> 侧边栏抽屉展示 Markdown 内容。
* Step 4: 引入 LocalStorage 记忆系统，实现连续上下文 Prompt 注入。
